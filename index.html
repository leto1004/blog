필요한 것들

Ruby(ruby, DevKit)
Jekyll
Python(Setuptool,pip,Pygments)
rouge
bundler

루비 설치

http://rubyinstaller.org/downloads/
Development Kit 설치 경로는 D:\RubyDevkit
CMD 창에서
cd C:\RubyDevkit
ruby dk.rb init
ruby dk.rb install

지킬(Jekyll) 설치하기

CMD 창에서
gem install jekyll
gem install rouge

python 설치

https://www.python.org/downloads/
윈도우 환경변수 설정
C:\Python27;C:\Python27\Scripts;
CMD 창에서
python
pip

Pygments 설치

CMD 창에서
pip install Pygments
설치 안되는 경우 컴퓨터 이름을 영문으로 변경

제킬(jekyll) 실행하기

CMD 창에서
mkdir C:\jekyll
jekyll
jekyll serve
gem install wdm
gem install bundler
