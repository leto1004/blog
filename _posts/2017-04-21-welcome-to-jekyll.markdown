## 드디어 Jekyll을 설치하다.
퍼블리셔로 html css javascript를 주로 다루고 있고 개발자를 꿈꾸고 있었지만 실제 개발자들이 많이 이용하는 GitHub는 거의 사용해보지도 않았고, 나에겐 너무나 어려운 개념들이 였다. (윈도우 CMD 조차 익숙하지 않았기에...) github를 통한 블로그는 쉽지 않은 도전이였다.

그럼에도 개인적인 생각과 혼자 공부중인 여러 IT관련 기술들을 정리할 공간은 블로그 만한 것이 없다고 판단했기에
이런 저런 블로그를 구글링으로 알아보던 중 우연하게 발견한 한 Jekyll 블로그를 보고 결심을 했다.
나도 해봐야 되겠다고. 언제까지 이렇게 미뤄둘순 없다고.. 정말 글쓰기에 집중하고 관리를 해보자고..

그렇게 Jekyll 기반의 GitHub Pages 블로그는 시작되었습니다.

Jekyll은 아래와 같이 코드뷰를 지원하는 것 같군요.

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
